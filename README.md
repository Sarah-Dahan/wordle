# Wordle Game

## :page_with_curl: About
This is an interactive word-guessing game with color-coded feedback and on-screen keyboard (an imitation of the New York Times Wordle game).  
This was a project I created while learning JavaScript foundations with Frontend Masters.

## :telescope: How To View and Play
Open in your browser with [this link](https://sarah-dahan.github.io/wordle/).

## :computer: Technologies
- HTML
- CSS
- JavaScript
